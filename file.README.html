Composite Unit Measurements

A set of specialized parsers for dealing with composite measurement strings.

Harshal V. Ladhe, M.Sc. Computer Science.

Introduction

The CompositeUnitMeasurements gem is a versatile solution tailored for parsing composite measurement strings. By harnessing the capabilities of the unit_measurements gem, it empowers you to seamlessly handle composite measurements in various units.
